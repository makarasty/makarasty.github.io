Перелік питань для самостійного опрацювання

Вступ до мультимедіа

  1. Стандарти в галузі мультимедійних систем.

    • Визначення єдиних норм і протоколів для зберігання, передачі та відтворення мультимедійних даних.
    • Приклади: ISO/IEC стандарти, MPEG, Plug and Play.

  2. Мультимедійні додатки, їх використання.

    • Програми, що інтегрують текст, звук, зображення, відео та анімацію.
    • Застосування: освіта, реклама, розваги, симулятори.

  3. Апаратні засоби мультимедійних систем.

    • Фізичні компоненти: відеокарти, аудіоадаптери, монітори, оптичні носії.
    • Характеристики: швидкодія, ємність пам’яті, підтримка різних форматів.

  4. Програмні засоби мультимедійних систем.

    • ПЗ для обробки та відтворення мультимедіа: кодеки, редактори, плеєри.
    • Приклади: Windows Media Player, Adobe Premiere, VLC.

Робота зі звуком та відео у мультимедійних системах

  1. Пристрої для роботи зі звуковою інформацією.

    • Обладнання: мікрофони, аудіоадаптери, динаміки, навушники.
    • Основні параметри: частота дискретизації, розрядність, АЦП/ЦАП.

  2. Методи і пристрої для створення спеціальних звукових ефектів.

    • Техніки: синтез звуку, семплювання, ефекти реверберації, ехо.
    • Пристрої: синтезатори, MIDI-контролери, аудіоінтерфейси.

  3. Підходи до стиснення відеоінформації.

    • Алгоритми стиснення: MPEG-2, H.264/AVC, VC-1.
    • Методи: внутрішньокадрове стискання, прогнозування кадрів, дискретне косинусне перетворення.

  4. Засоби відображення відеоінформації.

    • Пристрої: відеомонітори, телевізійні екрани, проектори.
    • Технології: LCD, LED, OLED, інтерлейсні стандарти.

  5. Апаратне та програмне забезпечення для роботи з відео.

    • Апаратне: відеокарти, декодери, спеціальні пристрої захоплення відео.
    • Програмне: відеоредактори, кодеки, мультимедійні плеєри (наприклад, Adobe Premiere, Final Cut Pro).

Комп’ютерна анімація

  1. Дванадцять принципів анімації.

    • Основні принципи: стискання і розтягнення, антіпод, перекладання, спільний рух, поступове появлення/зникнення, циклічність, акценти, таймінг, експресія, припинення руху, зв’язність, послідовність.

  2. Експортування анімації.

    • Формати файлів для збереження: GIF, AVI, MOV, MP4.
    • Параметри експорту: роздільна здатність, частота кадрів, компресія.

  3. Перекладна анімація. Створення перекладної анімації.

    • Техніка створення руху шляхом малювання послідовних змін об’єкта.
    • Застосування: традиційні мультфільми, покадрова анімація.

  4. Анімація обертання.

    • Принципи створення плавного обертального руху.
    • Інструменти для моделювання обертальних анімацій.

  5. Інструменти трансформації та деформації.

    • Методи зміни форми об’єкта: масштабування, обертання, зміщення, деформація.
    • Використання ключових кадрів для плавних переходів.

  6. Анімація кольору.

    • Зміна кольору об’єкта з часом для створення візуальних ефектів.
    • Техніки: градієнти, змінювані палітри, фільтри.

Основи motion дизайну

  1. Інструмент Da Vinci Resolve для створення та обробки анімації.

    • Функції: монтаж, кольорокорекція, робота з ефектами.
    • Основні можливості: інтеграція відео та аудіо, таймлайн-редагування.

  2. Засоби After Effects для створення та обробки анімації. Використання базових інструментів.

    • Основні функції: робота з шарами, ключовими кадрами, ефектами та трансформаціями.
    • Інтерфейс: таймлайн, панелі ефектів, композиції.

  3. Робота зі звуком в анімації.

    • Інтеграція аудіо до анімаційних проєктів.
    • Синхронізація звукових ефектів з візуальними елементами, використання аудіоредакторів.
